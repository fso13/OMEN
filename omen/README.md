# ORACLE/SHADOW — OMEN

Консольная игра в браузере (xterm.js): виртуальная ФС, сохранение в `localStorage`, деплой на GitHub Pages.

## Разработка

```bash
cd omen
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Переменная `GITHUB_PAGES_BASE` задаёт `base` в Vite (для project site: `/<repo>/`).

## Управление

- **Сохранить / Загрузить / Сброс** — кнопки над терминалом.
- **Ctrl+S** / **Cmd+S** — быстрое сохранение.

## Краткий путь

1. `ssh guest@main` → пароль `silencio-7` → серийник `K9F2` (музей `X7Q1` ведёт в honeypot).
2. На main: `decrypt-bundle` с паролем `ORACLE_SHADOW#2027_NULL`, `ssh internal.core` → `operator_7`.
3. `mirror` (два слова) → `mirror-bind` **или** на main `witness-pack` после `cat` токена на internal.

Подробнее: `docs/ORACLE_SHADOW_DEVELOPER_TZ.md` в корне репозитория.
